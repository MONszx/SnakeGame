# SnakeGame
Vanilla JavaScript Game with Boostrap UI

<h1> Instructions! </h1>
<p>1. Use ← ↑ → ↓ Arrow keys for the movements of the snake.</p>
<p>2. Green square is the snake. </p>
<p>3. Red square is the food to eat. </p>
<p>4. Lastly, Have fun ! </p>


# Implementation
* Vanilla JavaScript
* HTML
* CSS
* BOOSTRAP
